# Play Store Listing — en-US

This directory holds every asset Google Play Console needs for the FlowCast
listing. Layout follows the [fastlane supply metadata
spec](https://docs.fastlane.tools/actions/supply/) so future upload automation
with `fastlane supply` needs zero migration.

## Files

| Path                                  | Purpose                                          | Regenerated by        |
| ------------------------------------- | ------------------------------------------------ | --------------------- |
| `title.txt`                           | App name (≤30 chars)                             | Manual                |
| `short_description.txt`               | Tagline shown in search (≤80 chars)              | Manual                |
| `full_description.txt`                | Long listing copy (≤4000 chars)                  | Manual                |
| `video.txt`                           | Optional YouTube promo URL                       | Manual                |
| `changelogs/default.txt`              | "What's new" for latest release (≤500 chars)     | Manual, every release |
| `images/icon.png`                     | 512×512 hi-res icon                              | `npm run store:icon`  |
| `images/featureGraphic.png`           | 1024×500 banner — **manual**, see README next to it | Manual             |
| `images/phoneScreenshots/*.png`       | Phone screenshots (min 2, max 8)                 | `npm run store:screenshots` |

## Regenerating assets before a release

```bash
npm run store:all
```

Edit `changelogs/default.txt` with the release notes, then upload the listing
fields + AAB to Play Console.
